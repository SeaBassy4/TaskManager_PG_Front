# Task Manager - Frontend

A lightweight, responsive web interface for managing tasks. This is the frontend portion of the Task Manager project, designed to communicate with a custom Node.js/Express REST API.

## Features

- **View Tasks:** Automatically fetches and displays current tasks from the database.
- **Add Tasks:** Quick and easy task creation.
- **Delete Tasks:** Remove completed or unwanted tasks with a single click.
- **Dynamic UI:** Friendly empty-state messages when no tasks are present.

## Tech Stack

- HTML
- CSS
- Vanilla JavaScript (Fetch API)
- Deployment: **Vercel**

## Live Demo

- **Frontend:** https://taskmngrfront.vercel.app/
- **Backend API:** Hosted on Render
